**«Лавка волшебницы»** — это кроссплатформенное десктопное приложение для анализа фотографий ладони человека. Программа использует алгоритмы компьютерного зрения (оператор Собеля, фильтр Гаусса, пороговую бинаризацию) для выделения контуров руки и основных линий (жизни, сердца, ума). Приложение создано в развлекательных целях и демонстрирует возможности современных технологий обработки изображений на платформе .NET.

## ✨ Основные возможности

- **Загрузка изображений** — поддержка форматов JPG, JPEG, PNG, BMP.
- **Интерактивное выделение области (ROI)** — возможность мышкой выбрать фрагмент ладони для точного анализа.
- **Автоматическое выделение линий** — контур руки (зеленый) и основные линии (красный) рисуются поверх фото.
- **Улучшение контраста** — предварительная обработка изображения для лучшего обнаружения линий.
- **История предсказаний** — сохранение всех гаданий с миниатюрами обработанных фото.
- **Сохранение обработанных изображений** — все результаты сохраняются в папку `Assets`.

## 🚀 Запуск приложения на Windows через консоль

### Шаг 1. Установка .NET SDK

Для сборки и запуска приложения требуется **.NET 10.0 SDK**.

1. Перейдите на официальный сайт: [https://dotnet.microsoft.com/download](https://dotnet.microsoft.com/download)
2. Скачайте установщик **.NET 10.0 SDK** для Windows (x64).
3. Запустите установщик и следуйте инструкциям.
4. После установки откройте **Командную строку (cmd)** или **PowerShell** и проверьте установку:

```bash
dotnet --version
Если вы видите номер версии (например, 10.0.100), установка прошла успешно.

### Шаг 2. Клонирование репозитория

Склонируйте репозиторий с GitHub на ваш компьютер:
git clone https://github.com/yourusername/predskoz.git
cd predskoz

### Шаг 3. Восстановление зависимостей
Перед первой сборкой необходимо восстановить все NuGet-пакеты, указанные в файле predskoz.csproj:
dotnet restore

### Шаг 4. Сборка проекта

Выполните сборку проекта в режиме Release:
dotnet build -c Release

### Шаг 5. Запуск приложения
dotnet run -c Release

### Шаг 6 (Опционально). Публикация автономного EXE-файла

Чтобы создать автономный .exe файл, который можно запускать без установленного .NET SDK, выполните:
dotnet publish -c Release -r win-x64 --self-contained true -p:PublishSingleFile=true -o ./publish

После успешной публикации в папке ./publish появится файл predskoz.exe. Его можно скопировать в любое удобное место и запускать двойным щелчком мыши.
